# mockREACT

Sprint toDo9 do módulo 5 - Resilia Educação


## Pré-requisitos

- Node.js v.16.15.1
- NPM v.8.11.0

## Pacotes instalados

Abra o terminal/Powershell e rode os comandos abaixo:

Clonando o repositório:

```
git clone https://github.com/elicadv/sprint-mock.git
```

Entrando na pasta:

```
cd sprint-mock
```

Instalando todos os pacotes:

```
npm install
```

Rodando o projeto:

```
npm run dev
```
